# 실행순서
1. 현재 로컬 아이피(cmd)
(local-deployment 폴더까지 이동 후,
configure_lan_private_ip_windows.bat 실행)
cd local-deployment
.\configure_lan_private_ip_windows.bat

2. 도커 실행
cd local-deployment
docker compose up

3. 서버 실행
cd server
npm install
npm start

4. 클라이언트(front ukids 프로젝트) 실행
cd ukids
yarn dev

4. 화상통화
a. 로컬에서 접속
http://localhost:5173/

b. 다른 디바이스에서 접속
https://xxx-yyy-zzz-aaa.openvidu-local.dev:5443/

xxx-yyy-zzz-aaa는 아이피 주소 .대신 -로 변경
아이피 주소는 local-deployment의 .env파일 안 LAN_PRIVATE_IP에 있다.
